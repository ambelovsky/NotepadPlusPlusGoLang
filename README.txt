++++ Installation Instructions ++++
1. Extract go.npp.udl.zip
2. Open Notepad++
3. Open the "User-Defined Dialogue" (View->User Defined Dialogue...)
4. Click "Import"
5. Navigate to "go.npp.udl.xml" wherever you extracted the file
6. Click "Save As..." in the "User-Defined Dialogue"
7. Type "go"
8. Close the "User-Defined Dialogue"

++++ Usage Instructions ++++
1. Open the "Language" menu in Notepad++
2. Scroll to the bottom where you should now see "go" as a menu option, and click it
3. Begin typing in go
